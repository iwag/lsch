*less-scheme
てきとうなscheme処理系

*これはなに？
てきとうにつくったschemeもどき処理系です．scheme処理系ではありません．
schemeの部分実装というとかっこいいひびきです．
C++でかいてあります．

*特徴
+STLをめちゃくちゃつかってる
+超富豪プログラミング
+遅い
+ビルトインの演算子が少ない(+ - * =)
+SICP chap.4を読んでつくった
+いちおうfibonacciは動いた

*使い方
buildはmakeでいっぱつ（なはず．
$ make 
$ ./lesh
(+ 1 2)
3
(* 2 3)
6
(define (f n ) (if (= n 1 ) 1 (* n (f (- n 1 ) ) ) ) )
ok
(f 10) 
3628800
(define (fib n ) (if (= n 0 ) 0 (if (= n 1 ) 1 (+ (fib (- n 1 ) ) (fib (- n 2 ) ) ) ) ) )
ok
(fib 10) 
55
なんで綴じかっこにスペースがあいてるかというとそういう流儀ではなく，パーサがばかなんで2つ以上綴じかっこがつづくと動きません．
